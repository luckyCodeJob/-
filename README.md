# 项目简介
基于Vue.js和SpringBoot的办公用品直售系统是一个功能全面的电子商务平台，它为管理员、商家和普通用户提供了一个直观且易于使用的界面。系统分为用户前台和管理后台两个主要部分，以满足不同用户的需求。在管理后台，管理员可以高效地管理整个平台，包括商家管理模块，允许他们添加、编辑或删除商家信息；商品管理模块，用于上架、下架或更新商品信息；商品类别模块，帮助分类和组织商品，以便用户更容易找到所需商品；商品订单模块，使管理员能够跟踪订单状态，处理退货和退款请求；以及用户管理模块，用于监控用户活动和权限。前台则为普通用户提供了一个简洁的购物体验，他们可以浏览商品、查看详情、下单购买，并管理自己的订单。商家也可以通过前台查看自己的销售情况和客户反馈，以便更好地优化产品和服务。整个系统的设计旨在提供一个无缝的购物体验，同时确保后台管理的高效和安全。
